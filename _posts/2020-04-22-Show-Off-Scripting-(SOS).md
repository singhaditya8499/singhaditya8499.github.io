Today's blog post is a bit of show off.  

On the blog Dated 30 March, I said I would code daily and push it to my Github repository. But 3 days back, I thought that I should push all my accepted solution i.e. from the time I started solving problems over Codeforces. I visited Codeforces and found some 36 pages of data, where on each page there were around 50 entries. I started manually counting every accepted solution, but forgot the count in between. So I wrote a script to count it for me. I found that a total of 941 accepted solutions are submitted. It felt good. 

Next task was to extract the code and create a local .cpp file for each accpeted solution. I started doing it yesterday. I was not very familiar with Selenium, Python, Scraping or Browser Automation at the begining. I devoted around 15 hours spanning 2 days (in Lockdown) and finally wrote a fully working script. I extracted and created a total of 937 files and then pushed them to my repository. 

Today was a good day! Scripting a machine to do a task for you is very ecstatic and fulfilling.
